Hipertropi

Tutorial ↴

Terdapat beberapa hal yang perlu diketahui sebelum terjun kedalam dunia olahraga, salah satu yang paling utama adalah Kedisiplinan. Disiplin terhadap tidur, latihan, dan makanan.

Kami membagi dasar latihan menjadi 3:

  • ↪Sets: Kelompok repetisi/perulangan tanpa dijeda istirahat.
  • ↪Reps: Perulangan/repetisi gerakan yang dilakukan sesuai petunjuk.
  • ↪Beban(KG): Pilihlah beban angkatan sesuai dengan kebutuhan anda.
  • Masing - masing dari mereka memiliki keterhubungan misalnya Sets dipisahkan oleh waktu istirahat, Reps sendiri harus dilakukan sesuai dengan petunjuk, jika anda disarankan (6 - 10) maka anda harus menyesuaikan Beban agar pada Sets terakhir anda setidaknya bisa mencapai 6 dan tidak boleh lebih dari 10.

>>>

Dada:

Gerakan Majemuk: Incline bench press atau variasi lainnya (3 set dengan 6-10 repetisi dengan beban sedang, istirahat 2-3 menit).

Gerakan Isolasi: Pec fly atau variasi lainnya (3 set dengan 8-12 repetisi dengan beban ringan-sedang, istirahat 2-3 menit).

Triceps:

Overhead Cable Extension: 3 set dengan 6-10 repetisi dengan beban sedang - berat, istirahat 2-3 menit.

Pushdown: 3 set dengan 6-10 repetisi dengan beban sedang - berat, istirahat 2-3 menit.

Punggung:

Variasi Pulldown: 4 set (1 set pemanasan, 3 set utama dengan 6-10 repetisi dengan beban sedang - berat).

Variasi Row: 3 set dengan 8-12 repetisi dengan beban sedang.

Biceps:

Concentration Curl: 3 set dengan 8-10 repetisi dengan beban sedang.

Setelah 2 hari olahraga, tubuh memerlukan waktu untuk beristirahat agar sel otot dapat bertumbuh, pastikan mendapat nutrisi yang cukup juga.

Lateral Raises: 4 set dengan 12-15 repetisi dengan beban ringan.

Shoulder Press: 3 set dengan 10-12 repetisi dengan beban sedang - berat.

Rear Delt Fly: 3 set dengan 12-15 repetisi dengan beban ringan - sedang.

Squats: 4-5 set dengan 8-10 repetisi dengan beban sedang.

Leg Curl: 3 set dengan 10-12 repetisi dengan beban sedang - berat.

Leg Extension: 3 set dengan 12-15 repetisi dengan beban sedang - berat.

Biceps:

Concentration Curl: 3 set dengan 8-10 repetisi dengan beban sedang.

Triceps:

Overhead Cable Extension: 3 set dengan 6-10 repetisi dengan beban sedang - berat, istirahat 2-3 menit.

Pushdown: 3 set dengan 6-10 repetisi dengan beban sedang - berat, istirahat 2-3 menit.
